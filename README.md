# Agency Chaos Simulator

A game about the beautiful chaos of running a small creative agency. Manage projects, balance client satisfaction, team morale, and finances to survive 12 weeks.

## Play

Open `chaos-design-agency-game/index.html` in your browser.

## Features

- Project management with scope creep mechanics
- Team morale and burnout systems
- Client conversations and decision-making
- Office visualization (Mini Metro style)
- Multiple victory paths

## Tech

Pure JavaScript, HTML, CSS. No dependencies.
